Linked Font Awesome Icons with hover effects 

== Description ==

Social media inline icons with nice hover effect. We're using Font Awesome Icons v.4.4.0 (585 icons). You can see all icons on [Font Awesome Icons site](https://fortawesome.github.io/Font-Awesome/icons/ "Font Awesome Icons").



== Installation ==

This section describes how to install the plugin and get it working.


1. Upload brankic-icons-widget.zip to plugins, like any other plugin, or upload unzipped folder brankic-icons-widget to wp-content/plugins/
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Widgets and use Brankic Icons Widget in sidebar you want
